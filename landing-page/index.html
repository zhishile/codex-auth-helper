核心优势

为什么选择 Codex 登陆助手？

告别繁杂的命令行 Session 提取，为 Codex 量身定制的本地敏捷解决方案。

100% 纯本地运行

您的所有核心 Session 数据和凭证均保存在浏览器本地，插件绝不向任何第三方服务器上传任何数据，保障账号的绝对安全。

一键提取与智能解析

当您在浏览器中登录 ChatGPT 后，点击扩展即可自动提取 Session 凭证，并智能解析过期时间、用户认证级别等状态。

完美生成配置文件

自动将提取到的数据整理为 Codex 平台完全兼容的 auth.json 配置，提供一键复制与快速文件下载，即下即用。

精简工作流

只需三步，秒速唤醒 Codex

极致的自动化配置流程，助您摆脱手机号码登录验证与过期困扰。

01

登录 ChatGPT 网页

使用Chrome浏览器正常登录 ChatGPT，保持网页为在线活动状态。

02

激活插件 一键导出身份信息

点击浏览器右上角插件图标，智能解析身份信息之后点击“导出身份信息”按钮，将文件 auth.json 保存到本地。

03

载入 auth.json 到 Codex 配置目录

将 auth.json 移动到本机 Codex 配置目录中：

Windows: %USERPROFILE%\.codex\auth.json

macOS: ~/.codex/auth.json

答疑解惑

常见问题解答

关于 Codex 登陆助手，你想知道的解答都在这里。

为什么我用 Codex 登录时会要求输入手机验证码，而这工具能解决？
Codex 经常因为安全校验而触发手机验证机制。使用本插件可以先在 ChatGPT 官网完成常规登录，然后将其 Session 凭证提取为 auth.json。通过直接导入凭证，Codex 就能够跳过手机短信校验和各种盾，安全畅快地完成对接。
使用这个插件安全吗？会泄露我的 ChatGPT 账户 Session 吗？
极度安全。插件代码完全在本地浏览器沙箱中运行，无任何外部服务器同步接口，我们 100% 承诺数据不上云。本插件在 Github 上开发所有源代码，你可以随时查看并核对插件的打包源代码，绝对不收集任何敏感隐私。
我的身份验证信息能保持多久有效？
有效时间完全取决于 OpenAI ChatGPT 官方 Session 自身的生存时间。通常 Session 的有效期会长达两周至一个月。插件会在提取时对凭证中的 expires 字段自动进行毫秒级解析，并在模拟面板中提醒您具体还有多久失效。过期后您仅需再次轻轻一点击，重新覆盖即可。
支持哪些浏览器？
支持所有基于 Chromium 内核的浏览器，例如 Google Chrome, Microsoft Edge, Arc, Brave, Opera 等。用户只需开启浏览器的“开发者模式”，选择“加载已解压的扩展程序”并将包含 manifest.json 的根文件夹导入即可。
